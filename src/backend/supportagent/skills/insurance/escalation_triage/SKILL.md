# Escalation Triage

Use this skill when the user asks whether a case should be escalated or when the answer may be compliance-sensitive.

When this skill is active:
- Highlight ambiguity, source gaps, compliance-sensitive details, and cases that require specialist review.
- If sources are insufficient, use the configured refusal wording.
- Avoid making final legal, compliance, or claim-settlement decisions.
- State the concrete reason for escalation when one is supported by the sources.
